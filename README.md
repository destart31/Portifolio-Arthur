## 📝 Sobre o Projeto

Este projeto é meu portfólio profissional, desenvolvido para apresentar um panorama das minhas experiências acadêmicas e práticas na área de tecnologia. O site foi construído com foco em um design limpo, interações dinâmicas e uma estrutura de código organizada.

Originalmente desenvolvido com PHP para modularização, o projeto foi adaptado para ser implantado como um site estático em plataformas como a Netlify, utilizando um script de build para gerar os arquivos HTML finais.

---

## ✨ Recursos Principais

*   **Design Moderno:** Interface limpa com um tema escuro e detalhes em ciano.
*   **Efeito de Vidro (Glassmorphism):** Barra de navegação com fundo translúcido e desfocado.
*   **Navegação Aderente (Sticky):** A barra de navegação se fixa no topo da página durante a rolagem para fácil acesso.
*   **Animações e Transições:** Efeitos sutis em links, botões e cards para uma experiência de usuário mais fluida.
*   **Botão "Voltar ao Topo":** Facilita a navegação em páginas longas.
